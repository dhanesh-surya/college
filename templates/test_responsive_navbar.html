How to Use This Navbar

2. Add Required CSS:
<link rel="stylesheet" href="{% static 'css/responsive-navbar.css' %}"> <link rel="stylesheet" href="{% static 'css/navbar-enhancements.css' %}">
3. Include JavaScript:

4. Required Dependencies:
Bootstrap 5.3.2+ Font Awesome 6.4.0+ Tailwind CSS (optional)
Pro Tip: The navbar automatically switches between desktop and mobile versions at the 992px breakpoint. All hover effects are disabled on mobile for better touch interaction.
